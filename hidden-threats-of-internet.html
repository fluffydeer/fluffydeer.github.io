Hidden threats of the Internet

This is an educational vitrual reality game which showcases the risks of using Internet. This game is a part of a set where we had to fix bugs, improve immersion and convert desktop games into virtual reality. It was an univeristy team project finished in June 2021. This game was updated by me. The game is composed of three levels.

First level teaches about viruses which users can meet along their journey across the Internet. Descriptions of them are shown on the blackboard. After the start of the level the user needs to correctly categorize the viruses according to the questions.

Second level teaches about phishing in emails. User needs to recognize phishing signs in the emails and collect fish.

Third level shows the importance of having strong passwords by building walls around you and protecting you from the hostile enviroment.
